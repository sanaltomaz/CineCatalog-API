## Setup e Execução

### Pré-requisitos
1. Crie uma conta em [omdbapi.com](https://www.omdbapi.com)
2. Confirme seu email para ativar a API_KEY
3. Copie sua API_KEY

### Configuração
1. Crie um arquivo `.env` na raiz do projeto:
```
API_KEY=sua_chave_aqui
ENDERECO=http://www.omdbapi.com
```

2. Certifique-se de que as variáveis estão configuradas corretamente

### Execução
Execute a aplicação através da classe `OmdbApplication`:

A aplicação estará pronta para fazer requisições à API do OMDB.
